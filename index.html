Q&A

Three main differences between web1, web2 and web3:

  1. Interactive web - web1 wasn't interactive to the user and mainly used as read only source, while web2 and web3 were interactive and allowed uesrs to interact with each other (web2 used as read and write and web3 used as read, write and execute).
  2. Main purpose: Web1 is simply an information portal that users can passively receive information. Web 2.0 mainly used as collaboration and inforamtion sharing. Web 3.0 is the semantic web, where computers can interpret inforamtion and intelligently generate and distribute useful content.
  3. Focus: Web 1.0 - Company Focus: since the data that was available arrived from companies. Web 2.0 - Community Focus: sharing data, user's communication. Web 3.0 - Individual Focus: share knowledge and connect content through search and analysis.
 

The differences of searching the query: "side effect of medicines"

  • In Web 1.0 the results are macro content - so we'll probably get results of the specific words that were specified in the query or part of them. So there is a chance that we will get results that are relevant only for the word 'medicines', for example.
  • In Web 2.0 the results are micro content - so we can get the varies result of: links, images, audio, video and more that relates to the query. The problem is that the tagging process in manualy so the results depened on the user's tags.
  • In Web 3.0 the results are micro content, but the tagging process is automatic, this way the data that can be analysed is significant and the results can be more precise.
